Cole abaixo o código recebido de um colega para importar as configurações.
Como deseja importar?
⚠️ O Código do Sistema de cada empresa será sempre mantido — apenas as regras DE/PARA serão mescladas.
✅ Tudo funciona pelo próprio arquivo HTML — sem servidor, sem internet, sem login.

🔵 Como compartilhar:

  1. Configure o DE/PARA da empresa e banco
  2. Clique em "🔗 Compartilhar Configurações"
  3. Selecione o que exportar e clique em "Copiar Código"
  4. Envie o código por WhatsApp, e-mail ou Teams
  5. O colega abre o arquivo HTML, clica em "🔗 Compartilhar", aba "Importar Código"
  6. Cola o código e clica em "Importar"

🟡 Observações:

  • As configurações ficam salvas no navegador de cada computador
  • Quando atualizar as regras, gere e envie um novo código
  • Use "Backup Geral" para salvar tudo em arquivo JSON
  • Distribua também o arquivo .html para os colaboradores
